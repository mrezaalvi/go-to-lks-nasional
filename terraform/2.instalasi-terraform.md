# Panduan Setup Terraform di Windows
Untuk menginstal Terraform di Windows, cara yang paling umum dan mudah adalah dengan mengunduh file biner (.exe) secara langsung.

Berikut adalah langkah-langkah instalasi manual yang paling standar:

## 1. Unduh File Biner Terraform
1. Buka halaman unduhan resmi Terraform di developer.
    [hashicorp.com/terraform/downloads.](https://developer.hashicorp.com/terraform/install)

2. Cari bagian **Windows**.

3. Pilih arsitektur yang sesuai (biasanya **AMD64** untuk Windows 64-bit modern).

4. Klik **Download** untuk mendapatkan file `.zip`.

## 2. Ekstrak dan Letakkan File
Terraform tidak memiliki installer otomatis (seperti `.msi`), jadi kita harus meletakkannya secara manual:
1. Ekstrak file     `.zip`     yang baru saja diunduh. Anda akan mendapatkan satu file bernama `terraform`.exe.

2. Buat folder baru di drive `C:` agar mudah dikelola, misalnya: `C:\terraform`.

3. Pindahkan file `terraform.exe` ke dalam folder `C:\terraform` tersebut.

## 3. Daftarkan Terraform ke Environment Path
Agar perintah `terraform` bisa dipanggil dari folder mana saja di Terminal/PowerShell, Anda harus mendaftarkan lokasinya ke **System Environment Variables**:

1. Tekan tombol **Windows**, ketik "env", lalu pilih "Edit the system environment variables".

2. Di jendela yang muncul, klik tombol Environment Variables (di bagian bawah).

3. Di bagian System variables, cari variabel bernama Path, pilih, lalu klik Edit.

4. Klik tombol New di sisi kanan, lalu ketik lokasi folder tadi: C:\terraform.

5. Klik OK pada semua jendela untuk menyimpan perubahan.

## 4. Verifikasi Instalasi
1. Buka **Command Prompt** atau **PowerShell** yang baru (tutup yang lama jika masih terbuka).

2. Ketik perintah berikut:
   ```bash
   terraform -version
   ```
3. Jika berhasil, terminal akan menampilkan versi Terraform yang terinstal.
